Visual Fridge Scanner

Pixtral 12B returns ingredients with bounding box coordinates. Frosted-glass labels render on the photo; tap any unlabeled area to identify missed items. Collision avoidance keeps labels readable.

Cook Mode

Full-screen dark UI for the kitchen. G-Rams is a custom British chef voice (ElevenLabs). Web Speech API for "next", "repeat", and open-ended questions — answers in context, then back to the recipe.

Living Pantry

After cooking, Mistral infers what you used and marks items ok, low, or depleted. The next 3-day plan reflects your real pantry. Less waste, smarter suggestions.

Tech stack

Next.js 15 · Pixtral 12B · Mistral Large · ElevenLabs · Web Speech API · Instacart. No complex infrastructure — the loop runs in the app.
